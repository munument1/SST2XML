# SST to XML

SST to XML is a small Windows tool for converting xTranslator `.sst`
dictionaries into XML translation resource files.

## How to use

1. Run `SST2XML.exe`.
2. Add one or more `.sst` files.
3. Choose an output folder.
4. Enter a language pair, such as `en_ko`, `en_ja`, `en_ru`, or `fr_de`.
5. Click `Convert All`.

The tool supports batch conversion and works with SSU8/SSU9 SST files.
